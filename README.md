# Vite Tailwindcss playground

## Getting started
```bash
# clone the project
git clone https://github.com/seven1193/vite-tailwindcss.git

# enter the project directory
cd vite-tailwindcss

# install dependency
npm install

# develop
npm run dev
```
This will automatically open http://localhost:3000

## Build
```bash
# build for production environment
npm run build
```
